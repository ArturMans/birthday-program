# birthday-program
# Программа для работы с датой рождения

## Описание
Программа запрашивает у пользователя дату рождения и выполняет:
1. Определение дня недели
2. Проверку високосности года
3. Расчет текущего возраста
4. Вывод даты в формате электронного табло

## Запуск Python версии
```bash
cd python_version
python birthday.py
